The Wikidata Class Browser provides a tool to browse the taxonomy of Wikidata. In the search box you can find classes. In Browse Perspective you can get an overview about all classes and properties. The Datatype View provides information about datatypes and there properties.

The data for Wikidata Class Browser was extracted with Wikidata Toolkit
